# Чек-лист применения цепочки правил на ферме

> **Для:** Операторов, ветеринаров, менеджеров фермы  
> **Цель:** Пошаговое применение RULE-004 → RULE-002 → RULE-001 → RULE-003  
> **Версия:** 1.0 (2026-04-11)

---

## Общий принцип

```
PREVENTION → DETECTION → DECISION → TREATMENT
   │            │            │           │
   ▼            ▼            ▼           ▼
RULE-004    RULE-002    RULE-001    RULE-003
```

**Золотое правило:** Не пропускайте этапы. Каждое правило проверяет предыдущее.

---

## ФАЗА 1: PREVENTION (RULE-004) — Сухостойный период

### Когда применять

- **DIM:** -60 до отела (дальний сухостой)
- **DIM:** -21 до отела (ближний сухостой)

### Что делать

#### 1.1 Оценка при отеле

- [ ] Измерить BCS при переводе в сухостой (цель: 3.0-3.5)
- [ ] Зафиксировать вес

#### 1.2 Управление рационом

- [ ] Дальний сухостой: энергия 90-100% NRC, NDF >45%
- [ ] Ближний сухостой: постепенное увеличение энергии до 105-110%
- [ ] **Красный флаг:** BCS растёт во время сухостоя → снизить энергию

#### 1.3 Мониторинг

- [ ] BCS каждые 2 недели
- [ ] Разделять группы: дальний / ближний сухостой
- [ ] **Контрольная точка:** BCS при отеле должен быть 3.0-3.5

### Условия перехода к следующей фазе

```
Триггер: DIM >= 3 (после отела)
Состояние: Корова введена в группу свежих
Действие: Начать скрининг RULE-002
```

---

## ФАЗА 2: DETECTION (RULE-002) — Скрининг

### Когда применять

- **DIM:** 3-14 дней (оптимально: день 7)
- **Частота:** Однократно для всех коров
- **Повторно:** Если серый диапазон или подозрение

### Что делать

#### 2.1 Подготовка

- [ ] Проверить оборудование (фотометр для BHB)
- [ ] Заготовить журнал/шаблон CASE-INPUT
- [ ] Назначить ответственного

#### 2.2 Измерение

- [ ] Взять кровь (плазма/сыворотка) или молоко
- [ ] Измерить BHB
- [ ] Зафиксировать DIM
- [ ] **Важно:** Проверить клинические признаки!

#### 2.3 Классификация результата

| BHB     | Результат          | Действие                                             |
| ------- | ------------------ | ---------------------------------------------------- |
| < 1.0   | Норма              | Мониторинг, повторить через 7 дней при риске         |
| 1.0-1.2 | **СЕРЫЙ ДИАПАЗОН** | Повторить через 3-5 дней                             |
| 1.2-2.9 | **SCK обнаружена** | Переход к RULE-001                                   |
| ≥ 3.0   | **Тяжёлый случай** | **Экстренно: обход RULE-001 → клинический протокол** |

#### 2.4 Красные флаги (блокируют RULE-002)

- [ ] Ацетоновое дыхание → **Клинический протокол**
- [ ] Атаксия (шатается) → **Клинический протокол**
- [ ] Сильное угнетение → **Клинический протокол**

### Условия перехода к следующей фазе

```
Триггер: BHB >= 1.2 mmol/L (и < 3.0)
Состояние: RULE_002_TRIGGERED
Действие: Собрать данные для RULE-001
```

## ФАЗА 3: DECISION (RULE-001) — Принятие решения

### Когда применять

- **Условие:** RULE-002 обнаружил SCK (BHB 1.2-2.9)
- **Время:** Немедленно после получения результата BHB

### Что делать

#### 3.1 Сбор данных (структурированный ввод)

- [ ] BHB: ___ mmol/L
- [ ] DIM: ___ дней
- [ ] DMI фактический: ___ кг
- [ ] DMI норма: ___ кг
- [ ] DMI %: ___ % (рассчитать: факт/норма × 100)
- [ ] BCS текущий: ___
- [ ] BCS при отеле: ___
- [ ] BCS потеря: ___ (рассчитать)

#### 3.2 Оценка условий

**Hard Conditions (блокирующие):**

- [ ] Клинические признаки кетоза? → **Если ДА → Клинический протокол**

**Hard Conditions (для срабатывания):**

- [ ] BHB > 1.2? → Должно быть ДА
- [ ] DIM < 30? → Должно быть ДА

**Soft Conditions (для уровня уверенности):**

- [ ] DMI < 80%? → 1 балл
- [ ] BCS потеря > 0.5? → 1 балл

#### 3.3 Принятие решения

| Soft score | Вердикт                   | Действие                          |
| ---------- | ------------------------- | --------------------------------- |
| 0          | RULE_001_TRIGGERED_LOW    | Системная коррекция + мониторинг  |
| 1          | RULE_001_TRIGGERED_MEDIUM | Системная коррекция в течение 24ч |
| 2          | RULE_001_TRIGGERED_HIGH   | Системная коррекция немедленно    |

#### 3.4 Ключевое решение

**Вопрос:** Можно ли применять PG (RULE-003)?

```
Если:
  - BHB 1.2-2.9 (SCK подтверждён)
  - Нет клинических признаков
  - Корова ест (DMI > 50%)
  - AST < 500 (нет тяжёлого липидоза)
  
Тогда: ДА → RULE-003 применим

Если:
  - BHB > 2.9 ИЛИ клинические признаки ИЛИ DMI < 50%
  
Тогда: НЕТ → Эскалация к ветеринару
```

### Условия перехода к следующей фазе

```
Триггер: SCK confirmed (BHB 1.2-2.9) + нет блокирующих факторов
Состояние: RULE_001_TRIGGERED (любой уровень)
Действие: Начать RULE-003 (PG протокол) ПАРАЛЛЕЛЬНО с системной коррекцией
```

## ФАЗА 4: TREATMENT (RULE-003) — Лечение

### Когда применять

- **Условие:** RULE-001 разрешил PG
- **Немедленно:** Не откладывать

### Что делать

#### 4.1 Подготовка

- [ ] Проверить наличие пропиленгликоля (300 мл × 5 дней = 1.5 л)
- [ ] Проверить шприц/зонд для перорального введения
- [ ] Назначить ответственного за дачу

#### 4.2 Протокол PG

**Дозировка:**

- 300 мл/день
- Разделить на 2 приёма: 150 мл утром, 150 мл вечером
- Курс: 3-5 дней (стандарт), до 7 дней при BHB > 2.0

**Способ:**

- [ ] Перорально (через зонд или с кормом)
- [ ] **НЕ в/в!**
- [ ] **НЕ в рубец!**

**Расписание:**

| День | Утро   | Вечер  | Контроль            |
| ---- | ------ | ------ | ------------------- |
| 1    | 150 мл | 150 мл | Аппетит             |
| 2    | 150 мл | 150 мл | Аппетит             |
| 3    | 150 мл | 150 мл | BHB (если возможно) |
| 4    | 150 мл | 150 мл | Аппетит             |
| 5    | 150 мл | 150 мл | **BHB обязательно** |

#### 4.3 Мониторинг

**Ежедневно:**

- [ ] Аппетит (DMI оценка)
- [ ] Общее состояние (бодрость, подвижность)
- [ ] Стул (PG может вызвать понос — нормально)

**День 5 (обязательно):**

- [ ] BHB повторно
- [ ] Оценка результата

#### 4.4 Оценка результата

**Успех (завершить протокол):**

- [ ] BHB < 1.2 mmol/L
- [ ] Аппетит восстановлен
- [ ] Нет прогрессии к клиническому кетозу

**Частичный успех (продлить до 7 дней):**

- [ ] BHB снизился, но > 1.2
- [ ] Аппетит улучшается

**Неудача (эскалация):**

- [ ] BHB > 1.4 или растёт
- [ ] Появились клинические признаки
- [ ] **Действие:** Обратиться к ветеринару + пересмотреть по RULE-001

## ОБРАТНАЯ СВЯЗЬ (Feedback)

### Если RULE-003 не сработал

1. **Пересмотр RULE-002:** Верный ли был диагноз SCK?
2. **Проверка соблюдения:** Все ли дозы даны?
3. **Поиск сопутствующего:** Метрит? Мастит?
4. **Эскалация:** Комбинированная терапия

### Если много случаев требуют RULE-001

1. **Аудит RULE-004:** Что не так в сухостойном периоде?
2. **Проверить:** BCS при отеле, рационы, группировку

## БЫСТРАЯ СПРАВКА

### Критические точки

| Ситуация             | Действие                   | Правило             |
| -------------------- | -------------------------- | ------------------- |
| BHB < 1.0            | Мониторинг                 | —                   |
| BHB 1.0-1.2          | Повторить через 3-5 дней   | RULE-002 GRAY_ZONE  |
| BHB 1.2-2.9          | Системная коррекция + PG   | RULE-001 → RULE-003 |
| BHB ≥ 3.0            | **Экстренно к ветеринару** | **Обход правил**    |
| Клинические признаки | **Экстренно к ветеринару** | **Обход правил**    |

### Кому не давать PG

- [ ] Клинический кетоз
- [ ] BHB > 2.9
- [ ] Полный отказ от корма > 48ч
- [ ] AST > 500
- [ ] Без сознания / атаксия

## ВАЛИДАЦИЯ НА ФЕРМЕ

### Минимальный набор для тестирования

**Цель:** 10 кейсов через полную цепочку

**Распределение:**

- 6-7 успешных (BHB < 1.2 к дню 5)
- 2-3 частичных (улучшение, но BHB > 1.2)
- 1-2 неудачи (требовалась эскалация)

**Что фиксировать:**

1. Время от обнаружения до начала лечения
2. Соблюдение протокола PG (% доз)
3. Результат через 5 дней
4. Экономику (затраты vs прирост молока)
