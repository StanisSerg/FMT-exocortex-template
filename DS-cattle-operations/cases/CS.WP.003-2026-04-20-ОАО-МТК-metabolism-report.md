## 4. РЕКОМЕНДАЦИИ

### 🔴 Субклинический кетоз (BHB ≥ 1,2)
**Коровы:** 102, 107, 108

**Действие:**
- Проверить возможность приёма пропиленгликоля (RULE-003).
- Исключить тяжёлую форму: анорексия > 24ч, клинические признаки, гепатическая липидоза.
- При блокерах → экстренный протокол, не PG.
- Повторить BHB через 3–5 дней.

### 🟡 Серая зона (BHB 1,0–1,2)
**Коровы:** 106

**Действие:**
- Повторить BHB через 2–3 дня.
- Мониторинг съёма корма (DMI). При DMI < 80% → профилактический PG.

### 🔴 Гипокальceмия (tCa < 1,8)
**Коровы:** 205

**Действие:**
- Проверить клинические признаки молочной лихорадки (атаксия, рецамбенция).
- Клинические признаки есть → IV кальций немедленно.
- Нет клинических → оральный кальций + контроль Ca через 12–24ч.

### 🟡 Пограничная гипокальceмия (tCa 1,8–2,0)
**Коровы:** 202, 206

**Действие:**
- Повторить Ca через 24–48 часов.
- Проверить DCAD рациона сухостойных.

### 🟡 pH Close-Up: избыточно кислый
**Коровы:** 207, 208

**Действие:**
- Проверить DCAD рациона. Возможен анионный перебор.
- Риск снижения кормового поведения перед отёлом.

### 🟡 pH Close-Up: щелочной
**Коровы:** 202, 206

**Действие:**
- Усилить анионную диету (сульфат/хлорид магния/кальция).
- Целевой pH 6,0–6,5 снижает риск послеродовой гипокальceмии.
