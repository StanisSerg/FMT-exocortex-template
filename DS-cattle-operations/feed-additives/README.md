# Feed Additives / Кормовые добавки

> Оценка кормовых добавок на основе правил PACK-cattle-science

---

## Структура

```
feed-additives/
├── evaluations/           # Готовые оценки
│   ├── EVAL-001-...
│   └── EVAL-002-...
├── products/              # Инструкции производителей (PDF, MD)
│   └── {manufacturer}/
└── TEMPLATE-EVALUATION.md # Шаблон для оценки
```

---

## Методология оценки

### 1. Сбор данных
- Инструкция производителя
- Научные исследования (SoTA)
- Цена и доступность

### 2. Оценка по правилам PACK
Каждая добавка оценивается через призму существующих правил:
- Соответствует ли механизму?
- Есть ли противоречия?
- В каких условиях работает?

### 3. Экономический расчёт
- Стоимость курса
- Ожидаемый эффект
- ROI

### 4. Вердикт
- Применять
- Применять с оговорками
- Не применять
- Требует проверки

---

## Статусы оценок

| Статус | Описание |
|--------|----------|
| draft | Черновик, неполная информация |
| completed | Готова, принято решение |
| superseded | Устарела, есть новая версия |

---

## Связь с CASE и DL

Оценка добавки → применение → фиксация в CASE:
```
EVAL-001 → Применили на ферме → CASE-005 → DL-005
```

Если добавка подтвердилась — создаём правило в PACK.
