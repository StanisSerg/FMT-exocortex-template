# Извлечённые данные

> **Правило:** Все файлы в этой папке генерируются из `sources/`. Ручные правки допустимы для исправления ошибок OCR, но должны быть задокументированы в `notes:`.

## Структура

| Папка | Содержимое | Статус |
|-------|-----------|--------|
| `CNCPS-2025/` | Рекомендации по питанию (4 группы) | ✅ Готово |
| `NASEM-2021/` | Таблицы 21-1, 21-2, 21-3 | 🚧 Partial |

## Формат

- **YAML** — основной машиночитаемый формат.
- Каждый файл содержит блок `meta:` с источником, датой, статусом.
- Значения с диапазонами записываются как строки (`"0.5-0.7"`).
- `N/D` — значение не определено в источнике.

## Обновление

При обновлении PDF в `sources/`:
1. Переизвлечь данные с помощью `pdftotext`.
2. Сравнить с текущим YAML.
3. Обновить YAML, сохранив ручные правки в `notes:`.
4. Перегенерировать `reference/`.
