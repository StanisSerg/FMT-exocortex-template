# Валидация данных

> **Назначение:** Проверка корректности и согласованности извлечённых данных.

## Планируемые проверки

1. **Синтаксис YAML** — валидность всех `.yaml` файлов.
2. **Диапазоны значений** — нутриенты в физиологически допустимых пределах.
3. **Согласованность источников** — перекрёстная проверка CNCPS vs NASEM для совпадающих групп.
4. **Обязательные поля** — наличие ключевых нутриентов (Ca, P, Mg, MP, NE).
