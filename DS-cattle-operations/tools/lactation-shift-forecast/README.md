# Прогнозатор сдвига лактационной кривой

Лёгкий Streamlit-инструмент для оценки влияния изменений лактационной кривой на продуктивность стада.

## Быстрый старт

```bash
cd DS-cattle-operations/tools/lactation-shift-forecast
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

Откроется веб-интерфейс по адресу `http://localhost:8501`.

## Что делает

1. Берёт структуру стада: количество коров × DIM × паритет.
2. Строит базовую лактационную кривую по модели Wood для каждого паритета.
3. Применяет один из сценариев:
   - **+Y л на DIM X** — масштабирует кривую так, чтобы в выбранной точке удой вырос на Y литров.
   - **+Y л перед запуском** — масштабирует кривую относительно пикового удоя (стартовый потенциал).
4. Считает Δ дневного удоя стада, Δ 305-дневного удоя, дополнительную выручку и прибыль с учётом кормовых затрат.

## Структура

- `app.py` — интерфейс Streamlit.
- `models.py` — модель Wood, симуляция стада, сдвиги, калиброванные presets.
- `economics.py` — экономический расчёт.
- `validate.py` — валидация модели на реальных данных фермы.
- `scenario_zenchtc.py` — сценарии сдвига для данных Зенченко Ленинский.
- `data/` — примеры CSV со структурой стада.
- `requirements.txt` — зависимости.

## Валидация на реальных данных

Проверена на данных КТ Зенченко МТК Ленинский (CASE-002, 800 голов, 10 групп).

```bash
python validate.py
```

Результаты:

| Метрика | Baseline | После калибровки |
|---------|----------|------------------|
| Смещение (bias) | +6.4% | +1.5% |
| MAE | 2.46 л | 1.87 л |
| RMSE | 3.65 л | 2.97 л |

Группа 19 (соматика) исключена из калибровки — модель Wood описывает здоровых коров.
В интерфейсе доступен preset **«Зенченко Ленинский»** с калиброванными параметрами.

## Ограничения

- Упрощённая модель: не учитывает выгон, сезонность, индивидуальную вариацию, замену стада.
- Результат — оценка порядка величины для сравнения сценариев, не замена RuFaS/AMTS.
- Параметры Wood по умолчанию подобраны вручную; для точности калибруйте под реальные данные фермы.

## Пример

Стадо 90 коров:

| Паритет | DIM | Количество |
|---------|-----|------------|
| 1       | 100 | 20         |
| 2       | 100 | 30         |
| 3       | 100 | 25         |

Сценарий «+1,5 л на DIM 100» даёт прирост дневного удоя стада примерно на 100–120 л, что за год приносит дополнительно несколько миллионов рублей выручки (зависит от цен).
