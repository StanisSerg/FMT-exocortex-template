# Карточка сбора данных: 1 животное = 1 строка

> Метод структурированного сбора данных для животноводства.
> Source: inbox capture (2026-04-10)

## Принцип

**1 животное = 1 строка данных**

Не «анкета», а **инструмент** — система, из которой автоматически генерируются рекомендации.

## Структура карточки

### 1. Идентификация

| Поле | Описание |
|------|----------|
| Хозяйство | Название/код хозяйства |
| Группа | Учётная группа |
| ID животного | Уникальный идентификатор |
| Дата | Дата записи (YYYY-MM-DD) |

### 2. Возраст (критично)

| Поле | Описание |
|------|----------|
| Возраст (дни) | Точный возраст в днях |
| Категория | Физиологическая группа |

### 3. Возрастные категории (пример)

| Категория | Возраст | Особенности |
|-----------|---------|-------------|
| Тёлки 0-6 мес | 0-180 дней | Ранняя подсосная фаза |
| Тёлки 6-14 мес | 181-425 дней | Подращивание |
| Нетели 14-24 мес | 426-730 дней | Выращивание до отела |
| Дойные коровы | >730 дней | Лактация |

## Использование

1. **Заполнять один раз** — при первичном вводе животного
2. **Обновлять** — только дату и текущие показатели
3. **Ссылаться** — при анализе использовать ID для связи с другими данными
